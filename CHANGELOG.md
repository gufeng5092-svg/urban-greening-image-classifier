# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2026-06-29

- Added a reusable `urban_greening_classifier` Python package.
- Moved training, data and inference defaults into YAML configuration files.
- Added FastAPI inference endpoint at `POST /predict`.
- Added Dockerfile, docker-compose and GitHub Actions CI.
- Added pytest coverage for config loading, data loading, inference and utilities.
- Added project architecture documentation and contribution guidelines.
